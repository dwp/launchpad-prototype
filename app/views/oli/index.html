What it is

Demonstrates the latest version of the migration of the PIP service from authentication using DWP's Dynamic Trust Hub (DTH) to One Login. The prototype covers:

  • migrating DTH users to One Login
  • Signing in and signing up users via One Login
  • Post migration future state journeys
  • OIDV journeys
